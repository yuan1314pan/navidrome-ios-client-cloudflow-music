# Navidrome iOS Client

A modern, fast and offline-first iOS client for Navidrome.

## ✨ Features

- 🎧 Offline download (no network needed)
- 🏷️ Smart tags & AI-powered organization
- ⚡ Fast and smooth experience
- 📱 Native iOS design

## 🤔 Why this app?

Most Navidrome clients lack:
- Good offline experience
- Smart organization (tags)
- Smooth UI

This app focuses on solving these problems.

- Navidrome server
- Any Subsonic-compatible API
